SQLite добавление данных в одну БД из другой БД

Запуск SQLite CLI:

Запустить CMD и перейти в каталог sqlite driver:
    cd C:\Program Files\SQLite ODBC Driver for Win64
Запустить SQLite CLI:
    sqlite3.exe "C:\DB_catalog\database.db"

Операции в SQLite CLI:

Просмотр таблиц БД:
    .tables
Вывод списка полей таблицы в БД SQLite:
    PRAGMA table_info(table_name);
или:
    SELECT sql FROM sqlite_master WHERE type='table' AND name='table_name';
Присоедение второй БД SQLite данные из которой нужно взять:
    ATTACH DATABASE "C:\DB_catalog\database2.db" AS second_db;
Просмотр таблиц БД:
    .tables
Мержинг 
    INSERT INTO table_name (col1, col2, col3)
    SELECT col1, col2, col3
    FROM second_db.table_name;
Отследение второй БД SQLite:
    DETACH DATABASE second_db;
Очистка БД SQLite:
    vacuum;
Отключение от БД SQLite:
    .exit
