Windows CMD remote process management

## Get remote server process:

tasklist.exe /S IP_ADDRESS_SERVER /U USER_NAME_SERVER /P PASSWORD_SERVER
 

## Kill remote server process:

taskkill.exe /S IP_ADDRESS_SERVER /U USER_NAME_SERVER /P PASSWORD_SERVER /pid ....
 

## Get remote server user sessions:

quser /server:IP_ADDRESS_SERVER
 

or

qwinsta /server:IP_ADDRESS_SERVER
 

## Log Off remote server user by his session ID:

logoff ... /server:IP_ADDRESS_SERVER
