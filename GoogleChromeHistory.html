Google Chrome history

Google Chrome user profile history is accessible in local directory: 

AppData\Local\Google\Chrome\User Data\Default
in file with name "history" .
It is the SQLite datadase file. And the query to get the history is: 
SELECT
    url,
    title,
    visit_count,
    typed_count,
    datetime(last_visit_time / 1000000 - 11644473600, 'unixepoch', 'localtime') AS last_visit
FROM     urls
ORDER BY last_visit
