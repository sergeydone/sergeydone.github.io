### Postgresql собственный расчетный "коэффициент качества таблиц"

 

## Запрос для получения расчетного "коэффициента качества таблиц": 

	select last_autovacuum, relname, n_live_tup, n_dead_tup, autovacuum_count, * from pg_stat_user_tables order by /*autovacuum_count desc*/ last_autovacuum desc, relname
 
При настройках по умолчанию автовакуум срабатывает когда количество мертвых кортежей в таблицы превышает 20%, то есть чем больше коэффициент, тем более "раздута" таблица, и чем он ближе коэффициент к значению 0.20, тем скорее должен сработать автовакуум.
