Postgresql restoring inside cmd

Restoring from sql backup file 

cd /d E:\PostgreSQL\12\bin

psql.exe -h 127.0.0.1 -U postgres -d DATABASENAME -f C:\Backup\DATABASEBACKUP.sql > C:\Restore\restore_log.txt

Restoring from custom format backup file 

cd /d E:\PostgreSQL\12\bin

pg_restore -h 127.0.0.1 -v --username=postgres -d DATABASENAME "C:\Backup\DATABASEBACKUP.backup" 2>C:\Restore\restore_log.txt
