Find the Windows Client process which is connected to Postgresql Database.

# Inside PgAdmin or psql run: 

select * from pg_stat_activity
# Find the row you interesed in and writedown its fields: 'client_ip' and 'client_port'

# Connect into HOST with interested 'client_ip' and run in cmd:

netstat -ano | findstr :client_port
# writedown the process ID, and run:
powershell Get-Process -Id ID | Select-Object Name, Id, Path
