GSM modem REBOOT

## After successful connection to a GSM-modem with a terminal, you should type the AT-command:

at+cfun=1,1
 
 
# Your gsm-moden will go to reboot
# You might check modem status with AT-commands:
 
ati
 
at+csq
 
# First command returns gsm-modem information, and second command returns network quality
