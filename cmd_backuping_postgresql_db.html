Postgresql backuping inside cmd

Backuping into sql backup file 

cd /d E:\PostgreSQL\12\bin

pgdump -h 127.0.0.1 -p 5432 -U postgres -v BDNAME > DBBACKUPFILE.sql

Backuping into custom backup file format 

cd /d E:\PostgreSQL\12\bin

pgdump -h 127.0.0.1 -p 5432 -U postgres -F c -b -v -f "DBBACKUPFILE.backup" "DBNAME"
