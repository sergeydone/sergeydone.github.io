### Preparing VSCode for C# developing after installation

 

## 1 - install windows dotnet core 

	https://dotnet.microsoft.com/en-us/download
 

## 2 - creating first c# console app 

	dotnet new console -o app
 

## 2 - or creating first c# window app 

	dotnet new winforms -o app1
 

## 3 - testing created c# app 

	dotnet run
 

## 4 - download VSCode extension 

	https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp
 

## 5 - install via menu: 

	Extensions -> install from VSIX
 

## 6 - generate assets for build and debug in the command panel 

	dotnet run
