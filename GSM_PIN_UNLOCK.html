GSM modem sim-card pin unlocking

## After successful connection to a GSM-modem with a terminal, you should type a bunch of AT-commands:

Has sim-card blocked by the pin-code?

AT+CPIN?
 

Input correct puk-code and new pin-code if pin is blocked

AT+CPIN=MYPUCKCODEHERE,MYNEWPINCODEHERE
 

Check modem unlocking command

AT+CLCK=?
 

Disable pin-code checking

AT+CLCK="SC",0,"MYNEWPINCODEHERE"
 
# After all, you'll receive text messsage OK
