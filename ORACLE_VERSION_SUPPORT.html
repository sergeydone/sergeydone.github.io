Oracle Database 11g supported OS

## Существует следующая поддержка ОС со стороны Oracle Database 11g

 

# Oracle Database 11g Release 1 (11.1) for Microsoft Windows x64 официально поддерживает следующие ОС :

∙Windows Server 2003 - all x64 editions
∙Windows Server 2003 R2 - all x64 editions
∙Windows XP Professional x64 Edition
∙Windows Vista x64 - Business, Enterprise, and Ultimate editions 
 Источник: https://docs.oracle.com/cd/B28359_01/install.111/b32305/toc.htm#BGBFAAIC

# Oracle Database 11g Release 2 (11.2) for Windows x64 официально поддерживает следующие ОС :

∙Windows Server 2003 - all x64 editions
∙Windows Server 2003 R2 - all x64 editions
∙Windows XP Professional x64 Edition
∙Windows Vista x64 - Business, Enterprise, and Ultimate editions
∙Windows Server 2008 x64 - Standard, Enterprise, Datacenter, and Web editions
∙Windows Server 2008 R2 x64 - Standard, Enterprise, Datacenter, Web, and Foundation editions
∙Windows 7 x64 - Professional, Enterprise, and Ultimate editions
∙Windows 8 x64 - Pro and Enterprise editions
∙Windows 8.1 x64 - Pro and Enterprise editions

# Кроме того, Oracle Database 11g Release 2 начиная с версии 11.2.0.4 for Windows x64 официально поддерживает следующие ОС :

∙Windows Server 2012 x64
∙Windows Server 2012 R2 x64. (При этом Windows Server 2012 R2 x64 необходим Windows Database bundle patch 11.2.0.4.14 or higher) 
Источник: https://docs.oracle.com/cd/E11882_01/install.112/e47798/reqs.htm#NTDBI2698
